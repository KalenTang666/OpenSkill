## 1. 用户成长体系（Growth System）

### 1.1 钱包等级（Wallet Rank）

| Level | 名称 | 要求 | 解锁能力 |
|-------|------|------|----------|
| Lv.0 | Newcomer 🌱 | `oski init` | 基础 CRUD，1 个 Adapter |
| Lv.1 | Explorer 🔍 | 导入 3+ 资产 | `oski discover` + `oski analyze` |
| Lv.2 | Builder ⚡ | 创建 5+ 资产 + 1 次 scan | 全部 Adapters + `oski sync` |
| Lv.3 | Contributor 🤝 | 分享 1+ 到团队或市场 | Team 功能 + Marketplace 发布 |
| Lv.4 | Guardian 🛡️ | 签名 3+ 资产 + Trust Score ≥ 80 | Registry 发布 + 高级安全 |
| Lv.5 | Architect 🏛️ | 10+ 跨平台同步 + 社区贡献 | 自定义 Adapter + Commerce |

## 2. 商业化模型

### 2.1 Free → Pro → Enterprise 分层

| 功能 | Free | Pro ($9/月) | Enterprise (联系) |
|------|------|-----------|----------------|
| 平台适配器 | 3 | 全部 8 | 全部 + 自定义 |
| 资产数量 | 50 | 无限 | 无限 |
| 安全扫描 | 基础 (5 规则) | 完整 (14 规则) | 完整 + 自定义规则 |
| 团队成员 | — | 5 人 | 无限 |
| Marketplace | 浏览 | 浏览 + 发布 | 私有市场 |
| Registry | 本地 | 本地 + Hosted | 全部 + IPFS |
| 审计日志 | 7 天 | 90 天 | 无限 + 导出 |
| 成长体系 | Lv.0-2 | Lv.0-5 | Lv.0-5 + 定制 |
| 支持 | 社区 | 邮件 | 专属 + SLA |

### 2.2 Marketplace 分成模型

| 类型 | 创作者 | 平台 | 说明 |
|------|--------|------|------|
| 免费 Skill | 100% | 0% | 推广用 |
| 付费 Skill ($1-99) | 70% | 30% | 标准分成 |
| 企业 Skill ($100+) | 80% | 20% | 大额优惠 |
| 订阅制 Skill | 75% | 25% | 月度收入 |
